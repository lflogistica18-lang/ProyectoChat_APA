# 📋 Guía de Relevamiento de Datos — ProyectoChat_APA

> **Objetivo:** Esta guía sirve para completar la base de conocimiento del chatbot (Google Sheets). Utilizala durante la reunión con el cliente para asegurarte de tener toda la información necesaria.

---

## 1. Catálogo de Viandas (Imprescindible 🔴)
*Esta información permite que el chatbot describa qué vende el negocio.*

- [ ] **Menú/Carta actual:** ¿Tenés una lista, PDF o foto de lo que ofrecés hoy?
- [ ] **Descripción breve:** ¿De qué está hecho cada plato? (ej: "Milanesa de pollo con puré de papas real").
- [ ] **Opciones especiales:** ¿Cuáles son vegetarianas? ¿Cuáles son veganas? ¿Cuáles son Sin TACC (libres de gluten)?
- [ ] **Categorización:** ¿Tenés líneas diferentes? (ej: Línea Saludable, Línea Ejecutiva, Línea Económica).
- [ ] **Insumos gastronómicos:** Además de comida preparada, ¿vendés envases, bandejas o cubiertos descartables?

---

## 2. Política de Precios y Escalas (Prioridad Alta 🟡)
*Define cuánto cobrará el bot y qué descuentos aplicará por cantidad.*

- [ ] **Precio Base (B2C/Minorista):** ¿Cuál es el precio de una vianda para una persona particular?
- [ ] **Precio Negocio (B2B/Mayorista):** ¿Tenés un precio especial para revendedores o empresas?
- [ ] **Escalas por volumen:**
    - ¿A partir de cuántas unidades baja el precio? (ej: 10u, 50u, 100u).
    - ¿Cuál es el precio en cada escalón?
- [ ] **Impuestos:** ¿Los precios ya incluyen IVA o se suma después?
- [ ] **Vigencia:** ¿Hasta cuándo se mantienen estos precios o cuándo fue la última actualización?

---

## 3. Logística y Zonas de Entrega (Prioridad Alta 🟡)
*Permite que el bot confirme si el pedido puede ser entregado y cuánto cuesta el envío.*

- [ ] **Zonas de cobertura:** ¿A qué barrios o zonas llegás hoy? (CABA, GBA Norte, etc.)
- [ ] **Días y Horarios por zona:** ¿Entregás en todos lados todos los días? (ej: "Palermo: Lunes a Viernes 11-13hs").
- [ ] **Costo de envío:** ¿Cuánto sale el envío a cada zona?
- [ ] **Envío Gratis:** ¿A partir de qué monto o cantidad de viandas el envío es sin cargo?
- [ ] **Pedido Mínimo:** ¿Hay una cantidad mínima de viandas para que valga la pena el envío?

---

## 4. Promociones y Descuentos (Opcional 🔵)
*Información extra que el bot puede usar para incentivar la venta.*

- [ ] **Promociones fijas:** ¿Tenés promos tipo "Martes 10% OFF" o "Vianda del día"?
- [ ] **Cupones:** ¿Usás códigos promocionales en redes sociales?
- [ ] **Primera compra:** ¿Hay algún beneficio para quien compra por primera vez?

---

## 5. Datos Técnicos y Contacto (Para configurar ⚙️)
*Información para que el flujo de n8n funcione correctamente.*

- [ ] **Email de ventas:** ¿A qué dirección deben llegar las alertas de nuevos pedidos?
- [ ] **Telegram del equipo:** ¿Tenés un grupo de Telegram donde el bot deba avisar?
- [ ] **Dato B2B clave:** ¿A partir de qué volumen considerás que un cliente es "Gran Cuenta" (ej: +200 viandas)?
